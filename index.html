Efficient L-Shape Fitting for Vehicle Detection Using Laser Scanners 论文阅读与复现

发表于 2024-01-21 更新于 2024-01-27 分类于 Lidar Detection

解决的问题

解决使用旋转卡尺或者PCA分解生成错误heading角的问题。

论文基本思路

  1. 将90度n等分，旋转坐标系，计算所有点在新的方向上形成点的坐标
  2. 计算这些坐标值按照目标函数给出得分
  3. 选定得分最高的方向最为航向角方向
